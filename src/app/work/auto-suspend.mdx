## Auto Suspend

**Fungsi utama:**  
Mengatur proses dan layanan di Windows agar otomatis masuk mode suspend ketika idle.

**Penjelasan lengkap:**  
Auto Suspend membantu pengguna Windows mengurangi beban RAM dan CPU dengan cara mendeteksi proses idle lalu menaruhnya dalam kondisi suspend.  
- Mengurangi konsumsi RAM.  
- Memperpanjang umur battery laptop.  
- Menjaga sistem tetap responsif.  
- Ada mekanisme stepwise recovery jika proses penting ikut tersuspend.
