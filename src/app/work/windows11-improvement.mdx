## Windows 11 Some Improvement Stuff

**Fungsi utama:**  
Kumpulan command dan registry tweak untuk mengoptimalkan Windows 11.

**Penjelasan lengkap:**  
Project ini berisi beberapa script, command, dan registry modification yang bisa dipakai untuk:  
- **Performance boost:** tweak registry untuk mempercepat respon sistem.  
- **Disk & memory optimization:** command untuk mengurangi penggunaan resource berlebih.  
- **UI/UX improvements:** pengaturan tambahan agar Windows 11 lebih nyaman dipakai.  
- **Modular tweaks:** pengguna bisa pilih tweak sesuai kebutuhan, tidak harus semua.  

**Manfaat:**  
- Windows 11 lebih ringan dan responsif.  
- Memberi kontrol penuh pada pengguna untuk menyesuaikan sistem.  
- Cocok untuk power users yang ingin sistem lebih efisien.
